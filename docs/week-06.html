Grammar

가정법 (Conditional Sentences)

가정법은 사실과 다른 상황을 가정하여 표현하는 문법입니다. 직설법과의 차이를 이해하는 것이 중요합니다.

Grammar

1. 직설법 vs 가정법

직설법은 사실을 그대로 표현하고, 가정법은 사실과 다른 상황을 가정합니다.

As he is not rich, he can't go abroad. (직설법)
그는 부자가 아니므로 해외에 갈 수 없다. (사실 그대로)
If he were rich, he could go abroad. (가정법)
그가 부자라면 해외에 갈 수 있을 텐데. (사실과 반대)

Grammar

2. If 조건문의 4가지 유형

조건문은 상황에 따라 4가지로 나뉩니다.

Zero Conditional: If you touch a fire, you get burned.
불을 만지면 데인다. (일반적 사실)
If + 단순현재, 단순현재
First Conditional: If it rains tomorrow, I won't go to the beach.
내일 비가 오면 해변에 가지 않을 것이다. (가능한 미래)
If + 단순현재, will + 동사원형
Second Conditional: If I knew her phone number, I would call her.
그녀의 전화번호를 알면 전화할 텐데. (현재의 비현실 가정)
If + 과거, would + 동사원형
Third Conditional: If he had studied, he would have passed the exam.
그가 공부했더라면 시험에 합격했을 텐데. (과거의 비현실 가정)
If + 과거완료, would have + 과거분사

Grammar

3. If 생략과 도치

If를 생략하면 주어와 동사가 도치됩니다.

Were I you, I would not do such a thing.
= If I were you, I would not do such a thing.
were 도치
Had I any money, I would lend you some.
= If I had any money, I would lend you some.
had 도치
Should you need more information, please contact us.
= If you should need more information, please contact us.
should 도치

Grammar

4. If 대용 표현

If 대신 사용할 수 있는 다양한 표현이 있습니다: suppose, assuming that, otherwise, in your place, in case 등
