Grammar

가정법 과거완료 (Third Conditional)

과거 사실과 다른 가정을 할 때 사용합니다. 구조: If + 과거완료(had + 과거분사), would + have + 과거분사

If I had studied harder, I would have passed the exam.
내가 더 열심히 공부했더라면, 시험에 합격했을 텐데.
과거에 공부를 열심히 하지 않은 사실에 대한 가정
I would have been happy if you had invited me to the party.
네가 나를 파티에 초대했더라면, 나는 행복했을 텐데.
주절과 조건절의 순서가 바뀔 수 있음
If he had saved enough money, he would have bought a new car.
그가 충분히 돈을 모았더라면, 새 차를 샀을 텐데.
과거에 돈을 모으지 못한 사실에 대한 가정
If I had taken a taxi, I wouldn't have missed the train.
택시를 탔더라면, 기차를 놓치지 않았을 텐데.
부정형: would not(wouldn't) + have + 과거분사
