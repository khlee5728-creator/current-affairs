Grammar

부정대명사란?

특정한 사람이나 사물을 가리키지 않고 불특정한 대상을 나타내는 대명사입니다. (someone, anybody, everything, nothing 등)

Someone left their bag here.
누군가 가방을 여기 두고 갔다.
someone = 불특정한 사람

Grammar

some vs any

some은 긍정문에서, any는 부정문과 의문문에서 주로 사용합니다.

I have some money. (긍정문 - 존재하는 것)
나는 돈이 좀 있다.
긍정문에서 some 사용
I need to buy some groceries. (긍정문)
나는 식료품을 좀 사야 한다.
긍정문에서 some 사용
I don't have any money. (부정문 - "전혀 ~없는")
나는 돈이 전혀 없다.
부정문에서 any 사용
Are there any eggs? (의문문 - "있는지 확인")
달걀이 있나요?
의문문에서 any 사용
Any book will do. (긍정문 - "어떤 ~이라도")
어떤 책이든 괜찮다.
긍정문에서 any = "어떤 ~이라도"

Grammar

의문문에서 some 사용

권유나 부탁을 할 때는 의문문에서도 some을 사용합니다.

Would you like some coffee?
커피 좀 드시겠어요?
권유할 때 some 사용
Can you give me some sugar?
설탕 좀 줄 수 있어요?
부탁할 때 some 사용

Grammar

조건문에서의 any

조건문에서는 any를 사용합니다.

Let me know if there are any changes.
변경 사항이 있으면 알려주세요.
if 조건문에서 any 사용
