Practice: Expressions & Vocabulary

Grammar

3.1 부정대명사 — 개괄

부정대명사(Indefinite Pronouns)란 어떤 특정한 사람이나 사물을 가리키지 않고 막연한 대상을 가리킵니다.

  • 부정(Indefinite): 정해지지 않은, 분명히 규정되지 않은, 막연한
Lucy gave me this book.
루시가 이 책을 줬다.
책을 준 사람은 Lucy → 명확하다(definite)
Someone gave me this book.
누군가 이 책을 줬다.
책을 준 사람은 Someone → 명확하지 않다(indefinite) → 부정대명사

부정대명사는 매우 다양합니다:

All, another, any, anybody/anyone, anything, each, everybody/everyone, everything, few, many, nobody, none, one, several, some, somebody/someone

이 중 상당수는 부정대명사로 쓸 수도 있고, 부정형용사(한정사)로 쓸 수도 있습니다. 의미는 동일합니다.

I was just about to make some coffee.
나는 방금 막 커피 좀 끓이려고 했어요.
Some: 부정형용사(한정사) + 명사(coffee)
Some arrived at the camp.
일부는 캠프에 도착했다.
Some: 부정대명사

Grammar

3.1 some과 any — 기본 의미

some과 any의 기본적인 의미는 "조금, 약간"입니다. 적다는 의미라기보다는 불확실하다는 의미에 가깝습니다.

비슷하지만 some은 "조금 있다는 것을 가정할 때" 사용되며, any는 "있는지 없는지 확실하지 않을 때" 사용됩니다.

  • some: 긍정적, 존재할 때
  • any: 부정적, 존재하지 않거나 모를 때

그래서 보통 긍정문에서는 some을 사용하고, 부정문이나 의문문에서는 any를 사용합니다.

Grammar

3.1 긍정문에서의 some / any

긍정문에서 "조금, 약간"을 나타낼 때는 some을 사용합니다.

I have some money.
난 돈이 좀 있다.
some: 조금 있을 때
I need to buy some groceries before the store closes.
가게가 문을 닫기 전에 식료품을 좀 사야 한다.
some: 조금 사야 할 때

긍정문에서 any를 사용하면 "조금"이라는 의미가 아닌 "어떤 (것) 이라도"를 나타냅니다.

Any book will do.
어떤 책이라도 좋다.
Any: 어떤 ~ 이라도
You can choose any you like.
네가 좋아하는 어떤 것이라도 선택할 수 있다.
any: 어떤 것이라도

Grammar

3.1 부정문에서의 any

부정문에서 "조금, 약간"을 나타낼 때는 any를 사용해서 "조금의 ~도 없다"는 의미를 나타냅니다.

I couldn't find any.
나는 어떤 것도 찾을 수 없었다.
any: 조금의 것도(어떤 것도)
I don't have any money.
난 돈이 하나도 없다.
any money: 조금의 돈도
I didn't eat any food.
나는 음식을 조금도 먹지 않았다.
any food: 조금의 음식도
We should not make any noise in the library.
우리는 도서관에서 조금의 소음도 내지 말아야 한다.
any noise: 조금의 소음도

Grammar

3.1 의문문에서의 any / some

있는지 없는지 확실하지 않을 때 any가 사용되므로 의문문에서는 any를 사용합니다.

Are there any eggs?
달걀이 (조금이라도) 남아 있습니까?
있는지 없는지 확실하지 않을 때 → any
Did you buy any clothes in the shop?
가게에서 옷 좀 샀어?
있는지 없는지 확실하지 않을 때 → any

실제로 궁금해서 질문하는 것이 아니고 권유나 부탁할 때, 긍정적인 답변을 예상하거나 기대할 때는 some을 사용합니다.

Would you like some coffee?
커피 좀 드시겠습니까?
권유 → 긍정적 답변 기대 → some
Can you give me some sugar for my coffee?
제 커피에 설탕을 좀 주시겠어요?
부탁 → 긍정적 답변 기대 → some

Grammar

3.1 조건문에서의 any

조건문인 경우에도 "조금"을 나타낼 때는 some보다는 any를 더 자주 씁니다. "만약 ~이라면"의 의미이기에 통상적으로는 없을 가능성이 높기 때문입니다.

Let me know if there are any changes.
변경 사항이 있으면 알려주십시오.
만약 변경이 있다면 → 통상적으로는 없을 가능성이 더 많음 → any
If they need any help, tell them to call me.
만약 그들이 도움이 필요하면 저에게 전화하라고 말해 주세요.
만약 도움이 필요하다면 → 통상적으로는 없을 가능성이 더 많음 → any
